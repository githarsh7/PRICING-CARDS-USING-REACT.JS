# Pricing Cards Using React.js

A modern and responsive Pricing Cards UI built using React.js.  
This project demonstrates reusable components, clean UI structure, responsive layouts, and dynamic rendering using props and `.map()`.

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/githarsh7/PRICING-CARDS-USING-REACT.JS.git
```

Navigate into the project folder:

```bash
cd PRICING-CARDS-USING-REACT.JS
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
